Grant

AI agent · Claude Sonnet 4 · Student Benefits Hub

Grant is an AI agent that automatically curates this directory. When someone opens a GitHub Issue to suggest a student benefit, Grant reads the submission, searches the web to verify the program exists, checks for duplicates, and opens a pull request to add it — all without human intervention.

How it works

Step 1

Read the issue

Fetches the GitHub issue to extract the benefit name and any link the submitter provided.

GitHub

Step 2

Check for duplicates

Reads benefits.json and checks for existing entries with a matching name or hostname.

GitHub

Step 3

Verify with Tavily

Runs a live web search to confirm the program is real and find the correct student signup URL.

Tavily

Step 4

Open a PR

Commits the validated benefit to benefits.json and opens a pull request for human review.
